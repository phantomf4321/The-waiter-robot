## how to run

run A*:
```
$ python3 run.py a_star
```

run DFS:
```
$ python3 run.py dfs
```

run BFS:
```
$ python3 run.py bfs
```

run IDS:
```
$ python3 run.py ids
```

run UCS:
```
$ python3 run.py ucs
```
